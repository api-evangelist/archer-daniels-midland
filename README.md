# Archer Daniels Midland (archer-daniels-midland)
Archer Daniels Midland (ADM) is a global leader in agricultural processing and food ingredient manufacturing, headquartered in Chicago, Illinois. While ADM does not currently offer a widely publicized public developer API portal, the company leverages API integrations internally for supply chain management, commodity trading, and logistics operations.

### Commodity Data API
The ADM Commodity Data API represents ADM's data integration capabilities for agricultural commodity pricing, supply chain logistics, and product information. ADM processes and trades agricultural commodities globally, supporting partner integrations for commodity data access.
